# Map Pinning App
An app to save favourite locations on the map using react and leaflet.

## Configure the app
Create a .env file inside server folder and add MONGO_URL=your_mongo_url

## Install the app
for both client and server cd to the folder and run

```
pnpm i
```

## Run the app
for both client and server cd to the folder and run
```
pnpm start
```
